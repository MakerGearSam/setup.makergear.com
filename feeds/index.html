Octoprint Announcements

If you have not already, please make sure you have updated to Octoprint 1.3.6 and the MakerGear Setup Plugin 2.5.6.8 or later. To check your version of Octoprint and the Setup Plugin, click on the wrench icon in the navigatiopn bar, and go to 'Software Update' from left sidebar. You should see versions for both Octoprint and Makergear Setup.
