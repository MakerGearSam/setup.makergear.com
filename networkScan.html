MakerGear M3
3D Printer Scanner

This scanner requires Chrome or Firefox to work properly.

If you are using a different browser use the 'Use Manual IP Address" option

This tool will help you look for your printer if you do not have Bonjour/Zeroconf installed on your system. This might happen if you don't have the Bonjour Print Services installed on Windows, or if you are accessing your printer from a device that doens't support Bonjour/Zeroconf (like Android or Chromebooks). This scanner assumes that you have sucessfully moved the printer to your network (either via ethernet or via Wifi).

Warning

You have settings in Chrome that have disabled this tool.
  1. Copy the following link into a the Chrome address bar in a new window or tab. 
    chrome://flags/#enable-webrtc-hide-local-ips-with-mdns
  2. The top item should be “Anonymize local IPs exposed by WebRTC.”
  3. Use the drop down to set the value to ‘Disabled’
  4. At the bottom of the window, click 'Relaunch Chrome'
  5. Try using the print finder again.

If you are concerned about privacy issues related to IP addresses and WebRTC, when you are done using the print finder tool, you can follow the steps again, only changes 'Disabled' to 'Enabled'

You can also use the 'Manually Enter your Local IP' option which will bypass WebRTC / the chrome option in question. This will require you to know your local IP address, usually found in your Computer's Network Settings.
